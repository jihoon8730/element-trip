# 바가지킬러 · 바킬 (Bakil)

> 전국 관광지·지역 축제의 **실시간 물가 안심 지수**를 투명하게 제공하는 공정 관광 플랫폼

**2026 관광 데이터 활용 앱 공모전 출품작** · AI 시멘틱 크롤링 + 영수증 OCR 인증 기반

## 프로젝트 개요

매년 반복되는 지역 축제 야시장의 터무니없는 폭리(예: 1만 원 어묵, 5만 원 바베큐)와 카드 결제 거부 논란을 기술로 해결합니다.

한국관광공사 OpenAPI 위에 **AI 시멘틱 크롤링 엔진**과 **유저 영수증 OCR 인증 데이터**를 결합해, 소비자가 방문 전에 해당 관광지·축제의 물가 수준을 한눈에 확인할 수 있는 투명한 정보를 제공합니다.

## 시작하기

```bash
npm install
npx expo start
```

|---|---|
| iOS 시뮬레이터 | `i` |
| Android 에뮬레이터 | `a` |
| 실기기 | Expo Go 앱으로 QR 스캔 |
